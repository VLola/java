# Project 80.

### Task 1: 

Создать классы
- Продукт
- Категория
- Производитель

Настроить отношениями между ними

### Task 2: 

Создать коллекцию продуктов (10-15 продуктов, по 2-3 в категории)

### Task 3: 

Вывести:
- Все продукты
- Все категории
- Продукты от конкретного производителя
- Продукты ценой до 100$
- Продукты ниже средней цены в списке среди всех категорий
- Продукты ниже средней цены в списке среди самой НЕ популярной категорий
- Продукты самого не популярного производителя в самой популярной категорий
- Продукт весом до 1кг (случайный)
- Продукты ценой от 1$ до 5$ и весом до 500 грамм


### Task 4:

Записать все в файл формате json (имя файла - mydb.json)

### Task 5: 

Записать результаты работы 3-го блока в отдельный файл (имя файла - res.json)
